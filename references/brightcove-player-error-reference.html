Referencia de error de Brightcove Player

Este documento es la colección de errores de Brightcove Player. Tenga en cuenta que los errores se generan desde varios complementos diferentes.

Tipos de errores

Hay seis tipos principales de errores que se describen brevemente en esta sección. Puede hacer clic en el tipo de error para ver los errores específicos de cada tipo, si corresponde.

  • Errores estándar enviados desde el complemento de errores: Estos nombres de error comienzan con MEDIA_ERR.
  • Errores personalizados definidos por Brightcove y enviados desde el complemento de errores: Estos nombres de error comienzan con PLAYER_ERR.
  • Errores enviados desde el catálogo de Video Cloud: Estos nombres de error comienzan con VIDEO_CLOUD_ERR. Por supuesto, Jugador de Brightcove los clientes nunca verán estos errores.
  • Errores enviados desde el complemento HLS: Todos estos errores son del mismo tipo, APPEND_BUFFER_ERR , y el mensaje se recupera del error nativo del navegador.
  • Errores informados en la Consola de herramientas de desarrollo del navegador: Estos errores están orientados al desarrollador, ya que solo se verán en la consola del navegador.
  • Errores personalizados definidos por el usuario y enviados por el complemento de errores: Estos nombres de error los define el usuario, por ejemplo AGE_GATE_ERROR.
  • Errores de seguridad de la red

Errores estándar

En general, los errores de medios ocurren cuando surgen problemas al intentar recuperar o reproducir los medios. Por ejemplo, si se pierde la conexión a la red o si obtiene un video cifrado que no se puede descifrar, obtendrá un error multimedia. Para obtener detalles completos, consulte el Errores estándar sección de la Mostrar complemento de mensajes de error documento.

Código Tipo Titular Mensaje
1 MEDIA_ERR_ABORTED La descarga del video fue cancelada Abortó la reproducción multimedia.
2 MEDIA_ERR_NETWORK Se perdió la conexión de video, confirme que está conectado a Internet Un error de red provocó que la descarga de medios falle parcialmente. Actualmente es más útil para formatos de vídeo MP4 y/o descarga progresiva. Ver el Problemas conocidos sección de la Mostrar complemento de mensajes de error documento para obtener más detalles.
3 MEDIA_ERR_DECODE El vídeo es defectuoso o tiene un formato que no se puede reproducir en el navegador La reproducción multimedia se ha anulado debido a un problema de corrupción o porque las funciones multimedia utilizadas por su navegador no admiten.
4 MEDIA_ERR_SRC_NOT_SUPPORTED Este vídeo no está disponible o no es compatible con este navegador No se ha podido cargar el medio, ya sea porque el servidor o la red han fallado o porque el formato no es compatible.
5 MEDIA_ERR_ENCRYPTED El vídeo que intentas ver está encriptado y no sabemos cómo descifrarlo El video que está intentando ver está encriptado y no sabemos cómo desencriptarlo.
Desconocido MEDIA_ERR_UNKNOWN N/D Se encontró un problema inesperado, vuelva pronto e inténtelo de nuevo.

Errores personalizados

En general, los errores del reproductor ocurren cuando el video se ha cargado, por alguna razón no se puede reproducir. Por ejemplo, si las restricciones de dominio, geografía o IP impiden ver un video, se producirán errores en el reproductor. Para obtener detalles completos, consulte el Errores personalizados sección de la Mostrar complemento de mensajes de error documento.

Código Mensaje Descripción
-1 PLAYER_ERR_NO_SRC No se ha cargado ningún video.
-2 PLAYER_ERR_TIMEOUT No se pudo descargar el video, el tiempo de espera predeterminado es de 45 segundos.
No configurado PLAYER_ERR_DOMAIN_RESTRICTED Este video no se puede reproducir en su dominio actual.
No configurado PLAYER_ERR_IP_RESTRICTED Este video está restringido a su dirección IP actual.
No configurado PLAYER_ERR_GEO_RESTRICTED Este video no se puede reproducir en su región geográfica actual.

Errores de catálogo

En general, el catálogo arrojará errores cuando haya problemas para recuperar o reproducir inicialmente el video o la lista de reproducción. Por ejemplo, si usa una identificación que no existe, o una identificación de una cuenta diferente en la que existe el reproductor, para recuperar un video obtendrá un error de catálogo. Para obtener detalles completos, consulte el Errores de catálogo sección de la Catálogo de jugadores documento.

Error de catálogo Descripción
VIDEO_CLOUD_ERR_ACCOUNT_NOT_FOUND El reproductor se ha configurado con un ID de cuenta de Video Cloud no válido.
VIDEO_CLOUD_ERR_RESOURCE_NOT_FOUND El ID de lista de reproducción predeterminado no es válido.
VIDEO_CLOUD_ERR_VIDEO_NOT_FOUND El vídeo predeterminado no es válido.
VIDEO_CLOUD_ERR_NOT_PLAYABLE El video de Video Cloud no se puede reproducir.
Las causas frecuentes de este error incluyen cuando una identificación de video está desactivada o cuando la disponibilidad programada ha expirado.

Errores del complemento HLS

Para obtener detalles completos, consulte el Errores sección de la Complemento HLS documento.

Error del complemento HLS Descripción
APPEND_BUFFER_ERR El navegador arrojó un error al reproducir un video HLS.

Errores de consola

Estos errores aparecerían en una consola con un aspecto similar a este:

Mensaje de la consola Solución
VIDEOJS: ERROR: videojs-contrib-ads no ha visto un evento de inicio de carga 5 segundos después de inicializarse, pero hay una fuente. Esto indica que videojs-contrib-ads se inicializó demasiado tarde. Debe inicializarse inmediatamente después de video.js en la misma marca. Como resultado, algunos anuncios no se reproducirán y algunos eventos multimedia serán incorrectos. Para obtener más información, consulte https://github.com/videojs/videojs-contrib-ads#important-note-about-initialization Puede ver esto porque para que el complemento IMA3 funcione correctamente, debe cargarse justo después de la inicialización del reproductor, antes de que se inicialicen otros complementos. Ver el Implementar usando código sección de la Complemento IMA3 doc para la solución.

Errores de usuario personalizados

Para obtener detalles completos, consulte el Agregar errores personalizados sección de la Mostrar complemento de mensajes de error documento. No hay errores de usuario personalizados estándar.

Errores de seguridad de la red

Estos errores ocurren cuando la seguridad de su red bloquea algunos dominios y puertos de Brightcove.

Código de error Descripción
FLASHLS_ERR_CROSS_DOMAIN No se pudo cargar el video: acceso de dominio cruzado denegado.
Puede que necesite un crossdomain.xml archivo en la raíz de su archivo de lista de reproducción que permite el dominio que está solicitando (o '*').
