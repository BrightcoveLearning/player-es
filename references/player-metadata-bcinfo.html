Metadatos del reproductor de bcinfo

En este tema, aprenderá cómo recuperar información sobre el reproductor (metadatos) del bcinfo objeto.

Resumen de objetos

Aquí hay instancias específicas de console.log() y un console.table() vistas de los datos contenidos en el bcinfo propiedad.

Puede acceder a esta información utilizando un código similar al siguiente:

videojs.getPlayer('myPlayerID').ready(function() {
  var myPlayer = this;
  console.log('myPlayer.bcinfo', myPlayer.bcinfo);
  console.table(myPlayer.bcinfo);
});
los bcinfo el objeto se llena después videojs() se llama (o se ejecuta el script del reproductor). Entonces, usando la información del objeto junto con el ready() El método, como se muestra en el código anterior, es una práctica segura.

Propiedades del objeto

La bcinfo propiedad es un objeto que contiene información (metadatos) sobre el reproductor actual. Las propiedades presentes en el objeto son las siguientes:
