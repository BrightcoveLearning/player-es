Estilo de listas de reproducción

En este tema, aprenderá a diseñar listas de reproducción.

Resumen

Si desea modificar el aspecto de las listas de reproducción, puede hacerlo mediante CSS. En este documento verá varios estilos que puede cambiar. Por supuesto, los posibles cambios son numerosos y necesita encontrar el selector correcto para realizar su tarea.

Los siguientes documentos son información esencial para completar el panorama del uso de listas de reproducción:
  • Implementar listas de reproducción: Este documento muestra qué decisiones debes tomar sobre el uso de listas de reproducción y cómo obtener el código del reproductor que necesitas para obtenerlas en tu página HTML.
  • Plugin IU de lista de reproducción: Este documento detalla las opciones disponibles para mejorar la apariencia de la funcionalidad de la lista de reproducción.

El siguiente diagrama resume los estilos alterados para una lista de reproducción vertical. A continuación del diagrama hay una tabla que enumera los estilos.

  color: #E9D2B3;
}
Controla el color del
espacios entre elementos de la lista de reproducción		 .vjs-mouse.vjs-playlist {
  color de fondo: #161D6E;
}
Controlar el color del texto de duración .vjs-playlist .vjs-playlist-duration {
  color: #9BB4DA;
}
Controlar el color del fondo de duración .vjs-playlist .vjs-playlist-duration {
  color de fondo: #1160AA;
}
Control del espacio entre elementos de la lista de reproducción .vjs-playlist-vertical .vjs-playlist-item {
  margen superior: 20px;
}
Para evitar un margen por encima del primer elemento de la lista de reproducción, debe seguir el estilo de configuración de margen con lo siguiente: 
.vjs-playlist-vertical .vjs-playlist-item:first-child {
  margin-top: 0px;
}

Ocultar la barra de desplazamiento

No se necesita una barra de desplazamiento en la lista de reproducción horizontal, pero aparecerá. Si desea ocultar la barra de desplazamiento, el siguiente estilo hace precisamente eso:

div.vjs-playlist.vjs-playlist-horizontal {
  overflow-y: hidden;
}
