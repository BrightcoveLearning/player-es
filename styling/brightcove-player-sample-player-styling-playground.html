Muestra de Brightcove Player: Patio de juegos de estilo de jugador

Este tema le proporciona un campo de juego para experimentar con el estilo del jugador.

Patio de estilo

En este tema, utilizará un Codepen, que se muestra a continuación, como un campo de juego de estilo. Véalo en el codepen.io site para realizar cambios y siéntase libre de bifurcar el Codepen para su propio uso. Tenga en cuenta que es posible que desee ocultar el panel de JavaScript en Codepen, ya que aquí no se usa para nada, y agrandar el panel de CSS.

Ver la pluma Patio de juegos de estilo de jugador de Brightcove por Brightcove Learning Services (@bcls ) en CodePen.
