Personalización de la apariencia del jugador: aspecto heredado

Este documento contiene información sobre el aspecto de Brightcove Player 1.x (grafito), no sobre el aspecto más actual. Esta transición a la piel más nueva tuvo lugar la segunda semana de diciembre de 2015. Para obtener la versión más reciente del documento, consulte este documento.

Este tema es una introducción a la personalización de la apariencia del nuevo reproductor / styling / customizing-player-expression.html. El contenido utiliza las herramientas de desarrollo de Chrome, pero todos los navegadores modernos tienen una funcionalidad equivalente. Para obtener una breve introducción al uso de herramientas de desarrollo en diferentes navegadores, consulte la Conceptos básicos de depuración de JavaScript documento.

Crea tu propia piel

Si no quieres usar el aspecto predeterminado y crear el tuyo propio, esto es posible mediante la API de administración de jugadores. La Guía de configuración del reproductor proporciona la información necesaria.

Jugador

Nota: Este documento asume cierto conocimiento de CSS y el uso de las herramientas de desarrollo de un navegador para ver las clases de elementos HTML y examinar las CSS aplicables. Si no tiene el conocimiento básico de prerrequisito, sería mejor que trabajara a través del Inicio rápido: Personalización del jugador: aspecto heredado.

Puedes alterar la apariencia del jugador, pero primero debes tener una forma de abordarlo. Para ver esto, navegue por una página HTML con el código embed_in_page del reproductor insertado. En el Elementos sección de las Herramientas de desarrollo localice la primera <div> y verás que tiene una clase con el valor video-js asignados, entre otros.

Una vez que lo sepas, puedes usar estilos para alterar el propio reproductor. Por ejemplo, para cambiar el tamaño del reproductor, puede usar este estilo:

<style>
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

reproductor iframe

Si está utilizando la implementación de iframe del reproductor, la situación es diferente. Seguirá viendo al jugador con el video-js class, pero estará, por supuesto, dentro del iframe.

Los estilos que crees seguirán funcionando con el reproductor en el iframe, pero será necesario crear un archivo CSS y asociarlo con el reproductor mediante Studio. Haz esto yendo a la Jugadores módulo, luego haga clic en el video con el que desea asociar el CSS, luego en el Complementos sección agregue la ruta a su archivo CSS.

Si desea personalizar el iframe en sí, puede hacerlo agregando un class atribuir a la iframe elemento (línea 27). En este caso, el valor de la clase es videoFrame. Luego usa una <style> etiqueta para modificarla como desees. En el siguiente ejemplo, el width y height están alterados.

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Customize iframe</title>

  <!-- Page styles -->
  <style>
    .videoFrame {
      width: 640px;
      height: 360px;
    }
  </style>
</head>

<body>
  <iframe class="videoFrame" src='//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.html' allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
</body>

</html>

Botón de reproducción

Si desea modificar la apariencia del botón de reproducción, primero debe saber cómo abordarlo. En Chrome, asegúrese de Herramientas de desarrollo están abiertos y luego haga clic derecho en el botón. De las selecciones que aparecen, elija Inspeccionar elemento.

En el Elementos sección de las Herramientas de desarrollo, verá el código HTML que corresponde al elemento de botón.

En el código resaltado de arriba, puedes ver que la clase asignada al botón es vjs-big-play-button. El color y la opacidad del botón de reproducción (para que coincida con el tigre) se cambiarán con estos estilos:

  <style>
    .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Se podría pensar que este enfoque funcionaría, pero no lo hace. Al trabajar con CSS, debe tener en cuenta la especificidad de la propiedad. Esto hace referencia a cómo el navegador determina qué valores de propiedad se aplican a un elemento en función de su relevancia.

Una forma de aplicar los cambios de botón es hacer referencia a una identificación en el selector de CSS. Esto se debe a que los identificadores tienen una alta especificidad de CSS. Para obtener más detalles, consulte la Especificidad CSS sección

Primero, agregue un id atribuir a la div elemento para el código del jugador.

    <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

    <script src="//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.min.js"></script>

Luego, consulta el botón de juego grande del reproductor en el CSS de la siguiente manera:

  <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Este es el resultado de cambiar el botón de gris oscuro a naranja:

Botón de reproducción central

Puede centrar el botón de reproducción tanto horizontal como verticalmente utilizando los siguientes estilos:

top: 0px;
left: 0px;
right: 0px;
bottom: 0px;
margin: auto;

Ocultar botón de reproducción

Puede ocultar por completo el botón de reproducción utilizando el siguiente estilo:

display: none;

Una herramienta en Chrome es útil para verificar el elemento de nivel de volumen. Tenga en cuenta que solo verá el nivel de volumen cuando coloque el mouse sobre el botón de volumen. Para que se muestre el nivel de volumen al inspeccionar y encontrar el elemento correcto, siga estos pasos:

  • Haga clic derecho en el botón de volumen e inspecciónelo
  • Haga clic en el Botón de alternar elemento (resaltado en azul a continuación) en la parte superior derecha de la Estilos sección
  • Comprobar el :flotar cuadro (resaltado en rojo en el Estilos sección) para simular el mouse sobre mientras inspecciona los elementos
  • Profundice hasta encontrar el elemento de nivel de volumen (resaltado en rojo en la Elementos sección)

Ahora el background-color de estos dos elementos necesita un estilo. Quieres que tengan el mismo color que el botón, así que modifica tus estilos como se muestra aquí:

  <style>
    #video_1 .vjs-big-play-button {
      opacity: .6;
    }
    #video_1 .vjs-big-play-button,
    #video_1 .vjs-play-progress,
    #video_1 .vjs-volume-level {
      background-color: #B37D5B;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Tenga en cuenta que estamos anteponiendo cada clase con la identificación del jugador div elemento. Esto proporciona relevancia para los cambios de clase con respecto a la especificidad de CSS. De esto se hablará en la siguiente sección.

Controla la visibilidad

Puede dictar cuánto tiempo estarán visibles los controles. Usas el options().inactivityTimeout propiedad para hacer esto. Establece el valor igual a un número que es los segundos que los controles permanecen visibles después del final de cualquier actividad en el reproductor. Para que los controles permanezcan visibles todo el tiempo, establezca el valor en 0.

El siguiente código muestra la configuración del inactivityTimeout a 0 , lo que hará que los controles estén siempre visibles.

<video-js id="myPlayerID"
  data-account="1752604059001"
  data-player="04ee9f58-9d08-43ea-803f-abda86776db2"
  data-embed="default"
  class="video-js" controls></video-js>
<script src="//players.brightcove.net/1752604059001/04ee9f58-9d08-43ea-803f-abda86776db2_default/index.min.js"></script>

<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.options().inactivityTimeout = 0;
  });
</script>

Orden de los iconos de la barra de control

Puede controlar el orden de los iconos en la barra de control utilizando el order estilo asignado un número. Por ejemplo, 5 colocará un icono después de la barra de progreso pero antes del botón de volumen, y 9 colocará el icono en el extremo derecho de la barra de control. Para los navegadores más antiguos, es posible que se requieran algunos estilos más, como se muestra aquí:

{
  order: 5; /* after the progress bar but before the volume button */
  -webkit-box-ordinal-group: 5;
  -moz-box-ordinal-group: 5;
  -webkit-order: 5;
  -ms-flex-order: 5;
}

En el ejemplo de Brightcove Player se muestra un ejemplo de adición de un icono (para descargar un archivo en este caso): Descargue el documento del complemento de vídeo.

Especificidad CSS

La especificidad CSS hace referencia a la forma en que los navegadores determinan qué valores de propiedades se aplican a un elemento en función de su relevancia. Para obtener detalles sobre este concepto, revise el Especificidad artículo.

En Chrome, inspecciona el botón de reproducción grande. Tenga en cuenta que la background propiedad se establece mediante varias clases. Esto significa que no podemos usar solo la .vjs-big-play-button clase única para anular esta propiedad.

Aquí hay tres enfoques para personalizar el botón de juego grande:

  • Haz referencia a un identificador asignado al jugador.
  • Utilice varias clases para aumentar la relevancia de las propiedades.
  • Utilice JavaScript para establecer el atributo id en el reproductor y, a continuación, hacer referencia a él en el CSS.

Hacer referencia a un id

Dado que un id atributo tiene una gran relevancia con respecto a la especificidad de CSS, al agregar una id al selector de CSS se anulará el background-color propiedad.

Si aún no lo has hecho, añade un id atributo al video elemento de la siguiente manera:

    <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

En la CSS, agregue el valor id al selector de .vjs-big-play-button clase. Anular el color de fondo y la opacidad de la siguiente manera:

  <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Deberías ver que el botón de juego grande ahora está naranja.

Uso de varias clases

Desde el background La propiedad para el botón de reproducción grande se establece usando dos clases, necesitará al menos dos clases en el selector de CSS para anular esta propiedad.

Agregue la .video-js .vjs-big-play-button clase al selector de clases. Recuerde que la .video-js clase está configurada en el contenedor de vídeo.

Con estos valores, estamos diciendo que seleccione todos los elementos con un nombre de clase de vjs-big-play-button que son descendientes de elementos con un nombre de clase de .video-js.

<style>
  .video-js .vjs-big-play-button {
    background-color: #B37D5B;
    opacity: .6;
  }
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

Deberías ver que el botón de juego grande ahora está naranja. Trabajar con varias clases puede ser complicado y puede afectar a varios elementos, mientras que un id es específico de un elemento.

Uso de JavaScript

También puedes usar JavaScript para añadir dinámicamente un id al botón de juego grande. A continuación, puede hacer referencia a ese id desde el CSS.

Añade código para añadir un id al elemento del botón big play.

  • Línea 173: Obtiene una referencia al primer elemento que tiene una clase de vjs-big-play-button.
  • Línea 174: Añade un id atributo al botón de juego grande.
  <script type="text/JavaScript">

    var playButton = document.getElementsByClassName("vjs-big-play-button")[0];
    playButton.setAttribute("id", "myPlayButton");

  </script>

En el CSS, cambie el selector para utilizar el nombre de myPlayButton id.

  <style>
    #myPlayButton {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Deberías ver que el botón de juego grande ahora está naranja.
