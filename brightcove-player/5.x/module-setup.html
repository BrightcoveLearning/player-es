Módulo: configuración

setup.js: Funciones para configurar un reproductor sin interacción del usuario en función del atributo de configuración de datos de la etiqueta de vídeo.

Fuente:
setup.js que es

Métodos

<interno> autoSetup ()

Configura cualquier etiqueta que tenga un atributo de configuración de datos cuando se inicia el reproductor.

Fuente:
setup.js que es

<interno> autoSetupTimeout (espera [, vjs])

Espere a que se cargue la página antes de ejecutar la configuración automática. Esto se llamará en la configuración automática si HasLoaded devuelve false.

Parámetros:
Nombre Tipo Argumento Descripción
espera número

Cuánto tiempo esperar en ms
vjs videojs <opcional>

La función de biblioteca videojs

setup.js que es
