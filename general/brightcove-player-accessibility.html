Accesibilidad de Brightcove Player

{{ page.description }}

Controles de teclado

Brightcove Player se puede controlar mediante el teclado. Al desplazarse a los controles del reproductor, se pueden utilizar las siguientes teclas para controlar y manipular el reproductor:

  • Botón de reproducción grande: presione Espacio o Ingresar jugar
  • Botón de reproducción en la barra de control: presione Espacio o Ingresar para alternar entre reproducción y pausa
  • Barra de progreso: uso arriba o derecho botones para saltar hacia adelante 5 segundos, use abajo o izquierda botones para retroceder 5 segundos
  • Pantalla completa: presione Espacio o Ingresar para alternar entre pantalla completa y no pantalla completa
  • Control deslizante de volumen: uso arriba o derecho botones para subir el volumen, utilice abajo o izquierda botones para bajar el volumen

Lectores de pantalla

Brightcove Player es compatible con lectores de pantalla en navegadores de escritorio, así como con dispositivos iOS y Android.

Las pruebas más recientes están en contra de los estándares WCAG 2.0 AA utilizando lo siguiente:

  • TalkBack en la última versión de Android
  • VoiceOver en iOS 9+
  • Narrador de Windows con IE10 +
  • La última versión de JAWS en Windows con Firefox y Chrome en Windows
  • Mac OS VoiceOver con Safari, Chrome y Firefox
Cuando se utiliza el lector de pantalla JAWS con el navegador Google Chrome, es posible que los controles del teclado no funcionen como se esperaba. Mantener presionada la tecla Shift mientras usa los controles del teclado puede aliviar el problema. Sin embargo, se recomienda utilizar Firefox con JAWS en su lugar.
