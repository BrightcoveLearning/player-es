    Implementar listas de reproducción

    En este tema, aprenderá cómo utilizar listas de reproducción para mostrar una colección de vídeos agrupados en un orden determinado para su reproducción en un Brightcove Player.

    Resumen

    Al utilizar listas de reproducción, la primera decisión que debe tomar es dónde desea la lista de reproducción en relación con el jugador. Normalmente, al usar Brightcove Player se ven listas de reproducción a la derecha del reproductor verticalmente o por debajo del reproductor horizontalmente. (Como verá más adelante en este documento, tiene flexibilidad para ubicar la lista de reproducción en otros lugares). Las implementaciones de una lista de reproducción colocada vertical y horizontalmente siguen.

    /plugins/playlist-ui-plugin.html
    Los siguientes documentos son información esencial para completar la imagen del uso de listas de reproducción:
    • Complemento de interfaz de usuario de lista de reproducción: Este documento deta/styling/styling-playlists.htmlils opciones disponibles para mejorar el aspecto y la sensación de la funcionalidad de la lista de reproducción.
    • Estilo de listas de reproducción: Este documento proporciona información sobre cómo personalizar el aspecto de las listas de reproducción.

    Introducción a la implementación

    Para utilizar listas de reproducción, complete las siguientes tareas:

    1. Cree una lista de reproducción en Studio o manualmente usando JavaScript.
    2. Habilite a su reproductor para utilizar listas de reproducción mediante la opción de estilo de las propiedades del reproductor de Studio.
    3. Implemente la lista de reproducción utilizando el reproductor Estándar (iframe) o la implementación Advanced (incrustar en página) .
    4. Si desea controlar la lista de reproducción mediante programación, utilice la API de lista de reproducción.

    Activar reproductor

    Puede utilizar Studio para permitir que el reproductor utilice listas de reproducción. Siga estos pasos para hacerlo:

    1. Abra el módulo PLAYERS y cree un nuevo reproductor o localice el reproductor al que desea habilitar para las listas de reproducción.
    2. Haga clic en el enlace del reproductor para abrir las propiedades del reproductor.
    3. En Información del jugador, para el Tipo de jugador, haga clic en el botón de opción Lista de reproducción.
    4. Haga clic en Guardar.
    5. Haga clic en Estilo en el menú de navegación de la izquierda.
    6. Elija el tipo de lista de reproducciónque desee:
    7. Haga clic en Reproducir en el menú de navegación de la izquierda.
    8. Compruebe los botones de opción para ver las opciones de lista de reproducción que desea activar:
    9. Para publicar el reproductor, haga clic en Publicar e incrustar > Publicar cambios.
    10. Para cerrar el cuadro de diálogo abierto, haga clic en Cerrar.

    Reproductor estándar (iframe)

    Una vez que tenga una lista de reproducción para usar y haya habilitado a su jugador para usar una lista de reproducción, estará listo para utilizar realmente la lista de reproducción con un jugador. Como de costumbre, tienes disponibles las implementaciones de jugador estándar y avanzado. En esta sección, se muestran las instrucciones para usar la lista de reproducción con la implementación estándar. Cuando se utiliza la implementación de la lista de reproducción estándar o vertical, la lista de reproducción aparecerá en el lado derecho del reproductor, como se ha visto en el ejemplo de la parte superior de este documento.

    Publicar una lista de reproducción de manera muy similar a la publicación de un vídeo. En el módulo MEDIA de Studio, seleccione su lista de reproducción y haga clic en el botón Publicar lista de reproducción . Asegúrese de seleccionar el reproductor correcto.

    Una vez que haya publicado, elija su código. En esta sección, se muestra el código Estándar (iframe) . Puede seleccionar opciones para el código (resaltado) que se ocupan del tamaño del jugador.

    Si desea tener un reproductor receptivo, que es una excelente opción, el código del reproductor aparece como se muestra aquí:

      <div style="position: relative; display: block; max-width: 640px;">
    <div style="padding-top: 56.25%;">
      <iframe src="https://players.brightcove.net/1752604059001/S13cJdUBz_default/index.html?playlistId=5718313430001"
        allowfullscreen
        webkitallowfullscreen
        mozallowfullscreen
        style="position: absolute; top: 0px; right: 0px; bottom: 0px; left: 0px; width: 100%; height: 100%;"></iframe>
    </div>
  </div>

    Los dos div elementos y atributos asociados son los que hacen que el jugador responda.

    Ajustar padding-top para una lista de reproducción horizontal

    El código que se acaba de mostrar es sólo para el reproductor. Si está utilizando una lista de reproducción horizontal , debe ajustar manualmente el código un poco para dejar espacio para la lista de reproducción. Utilice la tabla siguiente para cambiar correctamente el <div style="padding-top: 56.25%;"> valor.

    Ración de Aspecto padding-top solo con reproductor padding-top con lista de reproducción horizontal
    4:3 75% 93,75%
    16:9 56,25% 70,31%

    Esto significa que si está utilizando un video de ración de aspecto 16:9, el padding-top valor debería aparecer de la siguiente manera:

      <div style="padding-top: 70.31%;">

    Cambio manual de la lista de reproducción

    Si desea cambiar la lista de reproducción utilizada con la implementación del iframe, simplemente necesita cambiar el src atributo del iframe. Si utiliza un ID:

      <iframe src="https://players.brightcove.net/.../index.html?playlistId=1754200320001"

    o si está utilizando un ID de referencia:

      <iframe src="https://players.brightcove.net/.../index.html?playlistId=ref:myrefid001"

    Reproductor avanzado (incrustado en página)

    Después de haber creado una lista de reproducción en el módulo MEDIA de Studio y haber habilitado un reproductor de listas de reproducción en el módulo PLAYERS de Studio, estará listo para implementar la lista de reproducción con un reproductor. Con el código de inserción estándar, el plugin crea el contenedor de la lista de reproducción y lo ajusta para ajustarlo al iframe elemento. Con el código de inserción avanzado en página, debe proporcionar el contenedor de la lista de reproducción y usar CSS para ajustar el tamaño del reproductor y la lista de reproducción de forma adecuada.

    Lista de reproducción vertical

    Para proporcionar una experiencia similar al ifram con una bandeja vertical plegable, Brightcove ha desarrollado un contenedor de reproductores de lista de reproducción para el código de inserción avanzado en la página.

    Para utilizar el contenedor del reproductor de lista de reproducción, ajuste el código de inserción avanzado con lo siguiente:

    <div class="vjs-playlist-player-container">
     

    Lista de reproducción fija, vertical

    Siga estos pasos para crear una lista de reproducción fija y vertical:

    1. Obtener el código de inserción avanzado. Después de publicar la lista de reproducción con el reproductor configurado (como se detalla en la sección anterior), elija el código Avanzado . Debería ser similar a esto:

      <video-js
  data-account="1752604059001"
  data-player="5dBwlwbNk"
  data-embed="default"
  controls=""
  data-video-id=""
  data-playlist-id="5686354139001"
  data-application-id=""
  width="960" height="540">
</video-js>

      Si has visto el reproductor con el código anterior, NO verás una lista de reproducción. Al utilizar la implementación avanzada del reproductor, DEBE envolver el reproductor en el contenedor de reproductores de la lista de reproducción.

    2. Envuelva el código de inserción avanzado con el contenedor del reproductor de lista de reproducción y agregue lo siguiente:

      • Un CSS estático width que coincida con el tamaño del jugador

      <div class="vjs-playlist-player-container" style="width: 960px;">
  <video-js
    data-account="1752604059001"
    data-player="5dBwlwbNk"
    data-embed="default"
    controls=""
    data-video-id=""
    data-playlist-id="5686354139001"
    data-application-id=""
    width="960" height="540">
  </video-js>
</div>
<script src="https://players.brightcove.net/1752604059001/5dBwlwbNk_default/index.min.js"></script>

      <video-js

    2. Envuelva el código de inserción avanzado con el contenedor del reproductor de lista de reproducción y agregue lo siguiente:

  <video-js
    data-account="1752604059001"
    data-player="5dBwlwbNk"
    data-embed="default"
    controls=""

      • Este porcentaje dividido por 0.8 nos da un porcentaje ajustado.

      • Multiplicar por 100 nos da un número de 0 a 100 que funcionará como un padding-top valor.

    Ejemplo: Envío del ID de vídeo para el iframe en la solicitud de página

    En el nivel más alto, la lógica detrás del siguiente código es un parámetro de consulta que se lee que contiene el ID de vídeo para el vídeo que debe reproducirse primero, luego el reproductor iframe se genera dinámicamente utilizando ese ID. Con más detalle, la lógica detrás de esta aplicación es:

    • Un punto de inserción se designa mediante la creación de un div elemento HTML.
    • El parámetro de consulta denominado PlayListVideoID se lee desde la URL de solicitud de página con ayuda de una función auxiliar.
    • Se construye dinámicamente un iframe elemento que incluye un src atributo que contiene el parámetro de consulta.
    • El iframe creado dinámicamente se inserta en la página.
        <!doctype html>
    <html>

    <head>
      <meta charset="UTF-8">
      <title>Untitled Document</title>
    </head>

    <body>

      <div id="place_player_here"></div>

      <script type="text/javascript">
        var theParamValue,
          iframe = document.createElement('iframe'),
          insertionPoint = document.getElementById("place_player_here");

        // Use the helper function below to get the value of the parameter
        theParamVideoID = getURLparam("playlistVideoId");

        // Dynamically build the iframe
        iframe.setAttribute('allowfullscreen', 'allowfullscreen');
        iframe.setAttribute('webkitallowfullscreen', 'webkitallowfullscreen');
        iframe.setAttribute('mozallowfullscreen', 'mozallowfullscreen');
        iframe.setAttribute('style', "width: 610px;height: 344px");
        iframe.src = 'https://players.brightcove.net/1752604059001/H1HpIEcCb_default/index.html?playlistId=4450721964001&playlistVideoId=' + theParamVideoID;
        // Insert the iframe into the page
        insertionPoint.appendChild(iframe);

        // Helper function to get value for a specific URL parameter
        function getURLparam(name) {
          var regex, results;
          name = name.replace(/[\[]/, "\\[").replace(/[\]]/, "\\]");
          regex = new RegExp("[\\?&]" + name + "=([^&#]*)");
          results = regex.exec(location.search);
          return results === null ?
            "" :
            decodeURIComponent(results[1].replace(/\+/g, " "));
        }
      </script>

    </body>

    </html>

    Playlist API

    La API de lista de reproducción le proporciona control programático sobre el uso de listas de reproducción, por ejemplo next(), previous() y autoadvance() métodos. La Guía: El documento de la API de lista de reproducción proporciona todos los detalles.

    Implementar usando código

    Si lo elige, puede implementar listas de reproducción completamente en código y evitar Studio. Los pasos básicos son:

    • Cargue el CSS y JavaScript para el complemento de interfaz de usuario de la lista de reproducción.
