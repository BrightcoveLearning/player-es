    Complemento Mostrar vistas previas de miniaturas

    En este tema, aprenderá sobre el plugin de vistas previas en miniatura que permite que la línea de tiempo para mostrar una imagen de vista previa en puntos definidos para su vídeo. El usuario puede pasar el ratón sobre la línea de tiempo y las imágenes definidas por usted se mostrarán para intervalos de tiempo especificados.
    Este plugin está obsoleto y ya no es compatible. Pronto llegará un nuevo plugin de miniaturas.

    Ejemplo

    La siguiente imagen muestra una miniatura que se muestra a medida que el usuario pasa el ratón sobre la barra de progreso:

    El siguiente vídeo muestra el uso del plugin de miniaturas. Empieza a reproducir el vídeo. A continuación, pase el ratón sobre la barra de progreso y seleccione una imagen en miniatura para saltar a esa sección del vídeo.

    Este artículo consta de los siguientes apartados:

    • Recopilar imágenes en miniatura
    • Implementación mediante el módulo Players
    • Propiedades del complemento de miniaturas
    • Uso de sprites de imagen CSS

    Recopilar imágenes en miniatura

    En este momento debe recopilar las imágenes usted mismo. Las miniaturas de vista previa se pueden generar como parte del proceso de codificación de Zencoder. Además, existen herramientas como SnapMotion para automatizar este proceso.

    No importa cómo genere las imágenes, debe decidir el número de imágenes que desea utilizar y elegir el tamaño adecuado.

    Implementar usando código de inserción

    Las miniaturas se pueden configurar mediante el complemento de miniaturas. Para implementar el complemento de miniaturas utilizando el código de inserción, siga estos pasos:

    1. En Video Cloud Studio, abra el módulo MEDIA .
    2. Seleccione el vídeo que desee utilizar y seleccione el botón Publicar e incrustar .
    3. En el cuadro de diálogo, seleccione el Reproductor predeterminado de Brightcove o un reproductor que haya configurado.

    4. Copie el código de inserción avanzado y agregue un id atributo a la video-js etiqueta. Debería verse similar a esto: 

      <video-js
  id="myPlayerID"
  data-account="1752604059001"
  data-player="default"
  data-embed="default"
  controls=""
  data-video-id="2026618970001"
  data-playlist-id=""
  data-application-id=""
  width="960" height="540"></video-js>
<script src="https://players.brightcove.net/1752604059001/default_default/index.min.js"></script>
    5. Pegue el código de inserción del reproductor en un editor con el código de página HTML.
    6. En la head sección del código de página, agregue el siguiente archivo CSS: 
      <link rel="stylesheet" type="text/css" href="https://players.brightcove.net/videojs-thumbnails/videojs.thumbnails.css">
    7. En la body sección, debajo del código de inserción del reproductor, agregue el siguiente archivo JavaScript: 
      <script src="https://players.brightcove.net/videojs-thumbnails/videojs.thumbnails.js"></script>

    8. Debajo del script anterior, agregue un bloque de script con lo siguiente:

      • Obtener una referencia al jugador cuando esté listo
      • Llame al thumbnails plugin con imágenes en miniatura de vista previa

    Propiedades

    Las siguientes propiedades del complemento se utilizan para controlar las imágenes en miniatura:

    • primer argumento:
      • El valor puede ser una cadena o un número, y está asociado a un objeto.
      • Esto define el tiempo en segundos que desea mostrar la imagen en miniatura.
      • Como mínimo, necesita una 0 propiedad con un src valor para cuando los usuarios se coloquen sobre el principio de la barra de progreso. Si tiene tiempos adicionales, el plugin particionará la barra de progreso y mostrará la imagen adecuada a medida que el usuario pasa el cursor sobre la barra de progreso.
       
      Nota: Debe incluir una 0 propiedad con un src valor. De lo contrario, el complemento intentará usar la miniatura de ejemplo y puede obtener un error.
    • src:
      • El valor debe ser una cadena.
      • Esta es la URL de la imagen en miniatura.
    • width:
      • Esta propiedad debe incluirse para la primera imagen en miniatura. Evita que la imagen se deslice fuera de la pantalla cuando se acerca al principio o al final de la barra de progreso.

    Aquí hay un ejemplo para configurar imágenes en miniatura durante tres bloques de tiempo en la barra de progreso:

    • La primera imagen se muestra cuando un usuario pasa el ratón por encima de la barra de progreso desde el inicio del vídeo hasta 5 segundos
    • La segunda imagen se muestra de 5 segundos a 10 segundos.
    • La tercera imagen se muestra desde 10 segundos hasta el final del vídeo
      {
    "0": {
        "src": "http://solutions.brightcove.com/bcls/assets/images/Tiger-4-preview.jpg",
        "width": 120
    },
    "5": {
        "src": "http://solutions.brightcove.com/bcls/assets/images/Water-4-preview.jpg"
    },
    "10": {
        "src": "http://solutions.brightcove.com/bcls/assets/images/BirdsOfAFeather-4-preview.jpg"
    }
  }

    Sprites de imagen CSS

    Si lo elige, puede usar sprites de imagen CSS como vista previa en miniatura. El siguiente JSON es un ejemplo para la options propiedad cuando se utiliza un sprite de imagen CSS:

    {
  0: {
    src: 'thumbnails.png',
    style: {
      left: '-60px',
      width: '600px',
      height: '68px',
      clip: 'rect(0, 120px, 68px, 0)'
    }
  },
  10: {
    style: {
      left: '-180px',
      clip: 'rect(0, 240px, 68px, 120px)'
    }
  },
  20: {
    style: {
      left: '-300px',
      clip: 'rect(0, 360px, 68px, 240px)'
    }
  },
  30: {
    style: {
      left: '-420px',
      clip: 'rect(0, 480px, 68px, 360px)'
    }
  },
  40: {
    style: {
      left: '-540px',
      clip: 'rect(0, 600px, 68px, 480px)'
    }
  }
