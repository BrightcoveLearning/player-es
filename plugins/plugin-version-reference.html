Usar en HTML

Si está utilizando las URL para los complementos directamente en una página HTML, debe usar un <script> etiqueta para JavaScript y una <link> etiqueta para el CSS. A continuación se muestran algunos ejemplos:

<script src="https://players.brightcove.net/videojs-vr/1/videojs-vr.min.js"></script>
<link href="https://players.brightcove.net/videojs-vr/1/videojs-vr.css" rel="stylesheet">
