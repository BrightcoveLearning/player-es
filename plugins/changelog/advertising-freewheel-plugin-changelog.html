    • Se ha corregido un problema de tamaño con los anuncios VPAID
    Nota: Como se mencionó en el documento del plugin real de FreeWheel, en la sección Using the correcto SDK URL , FreeWheel eliminó la URL de SDK dinámica a favor de una URL versionada. Esto significa que el plugin de Brightcove Player ahora está bloqueado a una versión específica de forma predeterminada, en lugar de la versión más reciente. Brightcove intentará mantenerse al día con las versiones de FreeWheel SDK, pero puede quedarse rezagada.

    Es posible que desee administrar su propia versión de SDK en su integración con Brightcove Player. Esto implica pasar la opción al plugin FreeWheel:Html5.sdkurl

    player.FreeWheelPlugin({
  Html5: {
    sdkurl: 'preferred-sdk-url'
  }
});
