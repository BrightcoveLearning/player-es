Registro de cambios del complemento de Chromecast

A continuación se realizan un seguimiento de los cambios realizados en el complemento:

Registro de cambios

8 de sep. De 2020

v2.0.0

  • Receptor Chromecast vuelto a implementar mediante la API de CAF

21 de mayo de 2020

v1.3.4

  • Integración de catálogo y SSAI para reproductor predeterminado

7 mayo 2020

v1.3.3

  • Corrección de errores: El botón no está visible si el anuncio de FreeWheel no se reproduce
  • Se corrigió un estado de carga persistente ocasional
  • Prueba de catálogo de receptor agregado

5 de marzo de 2020

v1.3.1

  • Se corrigieron las incompatibilidades entre el receptor Chromecast y Player v6.39 +

14 febrero 2020

v1.3.0

  • Corrección de errores: Evite errores de búfer de adición estableciendo la longitud máxima del búfer de objetivo en 30
  • Muestra el nombre completo del dispositivo en lugar de la primera letra
  • README actualizado con nuevos procedimientos de prueba
  • Se actualizó el archivo README con pasos para pruebas de contenido inseguras.

27 ago. 2019

v1.2.0

  • Traducciones añadidas para chino, alemán, español, francés, japonés y coreano

v1.1.5

v1.1.4

20 feb. 2019

v1.1.2
