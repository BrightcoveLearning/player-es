Nota: De forma predeterminada, las actualizaciones se aplican automáticamente a todos los reproductores. Para evitar la actualización automática de un reproductor, debes configurarlo en actualización manual (también conocido como bloqueo de un reproductor). Puede hacerlo en el módulo Studio Players (sitio externo) o mediante la API de administración de reproductores.

La exclusión de las actualizaciones automáticas debe ser una medida temporal, aplicada mientras resuelve cualquier problema que cree la actualización. Si no actualizas los reproductores a la versión actual, te estás perdiendo nuevas funciones y correcciones de errores, y las versiones anteriores del reproductor dejarán de ser compatibles en algún momento

Para obtener notas de la versión de reproductores anteriores, consulte el archivo.

Para obtener notas de la versión de complementos anteriores, consulte el páginas de complementos.
