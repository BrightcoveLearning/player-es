Prueba de complementos publicitarios previos al lanzamiento

{{ page.description }}

Resumen

Mostrar anuncios de video en Brightcove Player puede ser una tarea compleja. Varias entidades (Brightcove Player en sí, un servidor de anuncios y el complemento de anuncios específico) están involucradas en la entrega exitosa de publicidad. Para proporcionar un entorno de prueba para las actualizaciones de los complementos IMA3 y FreeWheel, Brightcove pondrá a disposición versiones preliminares de los complementos. Este documento proporciona instrucciones sobre cómo realizar pruebas con estas versiones preliminares.

Los prelanzamientos se anunciarán en el Foro de desarrolladores de Brightcove Web Player , así como en el registro de cambios secciones de la Publicidad con el complemento IMA3 y Publicidad con el complemento FreeWheel documentos.

Prueba de IMA3 en Studio

Los siguientes pasos lo guían sobre cómo probar un complemento IMA3 de versión preliminar en Studio:

  1. Por motivos de seguridad, debe registrar la versión actual / plugins / plugin-version-reference.htmls del complemento que está utilizando actualmente. Esto le permitirá restaurar la versión del complemento que estaba usando antes de realizar la prueba. Siempre puede encontrar la última versión del complemento en el Referencia de la versión del complemento documento.
  2. Abre el JUGADORES módulo y cree un nuevo reproductor o localice el reproductor al que desea agregar el complemento.
  3. Haz clic en el enlace del reproductor para abrir las propiedades del reproductor.
  4. Hacer clic Publicidad en el menú de navegación de la izquierda.
  5. Desmarque la Habilitar el lado del cliente (IMA). No desea que esto esté habilitado, ya que puede interferir con la versión que pronto cargará manualmente.
  6. Haz clic en Guardar.
  7. Haz clic en Plugins en el menú de navegación izquierdo.
  8. A continuación, haz clic en Agregar un plugin.
  9. Para el nombre del complemento, introduzca ima3.
  10. Para el JavaScript URL, ingrese la URL del archivo JavaScript de versión preliminar.
  11. Para el CSS URL, ingrese la URL del archivo CSS de la versión preliminar.
  12. Ingrese las opciones de configuración en el Opciones (JSON) caja de texto. Si ha configurado previamente el complemento IMA3 utilizando el Complementos sección, lo más probable es que no necesite realizar ningún cambio en las opciones.
  13. Haz clic en Guardar.
  14. Para usar el reproductor, haga clic en Publicar e insertar . Podrás probar usando el Reproductor de vista previa , o puede publicar y utilizar el Reproductor publicado.
  15. Para cerrar el cuadro de diálogo abierto, haga clic en Cerrar.

Prueba de la implementación programática

Probar dónde ha implementado el complemento mediante programación es el mismo si está utilizando el complemento IMA3 o FreeWheel. Básicamente, simplemente reemplaza las rutas existentes a los archivos JavaScript y CSS por las de la versión preliminar. A continuación se indican los pasos para realizar esta tarea.

  1. Por seguridad, debes registrar las versiones actuales del plugin que estás utilizando actualmente. Esto le permitirá restaurar la versión del complemento que estaba usando antes de realizar la prueba. Siempre puede encontrar la última versión del complemento en el Referencia de la versión del complemento documento.
  2. En su código, ubique el <link> y <script> etiquetas que extraen CSS y JavaScript para el complemento adecuado. En el código siguiente, se resaltan ejemplos del complemento IMA3.
  3. Reemplace esas rutas con las versiones preliminares.
  4. Actualice su página y estará usando la versión preliminar.
  5. Una vez completada la prueba, vuelva a editar las rutas a los archivos JavaScript y CSS utilizando las rutas registradas para restaurar su configuración original.
