Creando un Charles Log

{{ page.description }}

Charles es un proxy web que se ejecuta en su computadora y le permite registrar datos que se envían y reciben entre su navegador web y el servidor. Utilizando Charles facilita el diagnóstico y la solución de problemas. Para obtener más detalles, consulte la Charles proxy web sitio.

Cuando usas Charles , cierre tantas aplicaciones como sea posible y cierre todas las ventanas innecesarias del navegador para limitar la cantidad de información que captura Charles.

Pasos para crear un Charles Iniciar sesión

Los siguientes pasos le guiarán a través del uso del Charles proxy web para crear un registro grabado.

  1. Descargar e instalar Charles en su máquina. Charles es compatible con Windows, Mac y Linux. Para obtener detalles específicos de la plataforma sobre la instalación Charles , mira el Charles Instalación notas.

  2. En tu computadora, ejecuta Charles. En el menú, seleccione Proxy> Configuración de proxy. Asegúrese de que se haya asignado un puerto (por ejemplo, 8888) y Habilitar el proxy HTTP transparente está seleccionada la opción.
  3. Seleccione OK para guardar los cambios y cerrar el diálogo.
  4. Instala el Charles Certificado SSL. En el menú, seleccione Ayudar> Proxy SSL> Instalar en pc Charles Certificado raíz.
  5. En el Agregar certificado diálogo, seleccione Agregar.
  6. Acceso al llavero abrirá. En la lista de certificados, busque el Charles Apoderado... certificado. Haga doble clic en él para ver información detallada.
    Si no está utilizando macOS, consulte la Charles documentación para obtener detalles sobre los certificados SSL.
  7. Ampliar la Confianza sección. Para el Al usar este certificado opción, seleccione Siempre confía.
  8. Cierre la ventana del certificado e ingrese su contraseña de administrador para actualizar la configuración.
  9. Habilitar el proxy SSL en Charles. En el menú, seleccione Proxy> Configuración de proxy SSL. Asegurar la Habilitar el proxy SSL está seleccionada la opción. En el Localización sección, seleccione la Agregar botón.
  10. En el Editar ubicación cuadro de diálogo, navegue hasta el Puerto campo e ingrese 443. Seleccione OK para guardar este cambio.
  11. Seleccione OK para cerrar el Editar ubicación diálogo.
  12. Seleccione OK para salir del Configuración de proxy SSL diálogo.
  13. Se recomienda que reinicie su computadora en este punto para asegurarse de que Charles recoge la nueva configuración del certificado.
  14. Guardar Charles ábralo mientras realiza los siguientes pasos.
  15. Pegue la URL en la página que contiene el reproductor de Brightcove que está solucionando problemas en la barra de direcciones del navegador y, a continuación, presione Entrar para cargar la página.
  16. Ejecute los pasos que utilizó anteriormente para reproducir el error o problema. Charles registrará automáticamente lo que está sucediendo.
  17. Si se necesita tráfico de red para cuando el reproductor está reproduciendo un vídeo, inicie el vídeo.
  18. Después de reproducir el error, vuelva a Charles. Selecciona el Estructura ver y buscar la URL de su reproductor.
  19. Seleccione Para de grabar.
  20. Guarde la sesión actual en un archivo.
  21. Envíe un correo electrónico al Charles archivo de sesión al personal de soporte apropiado.

Agregar encabezados de Akamai

En esta sección, aprenderá a agregar encabezados de Akamai a sus solicitudes.

  1. Desde el principal Charles ventana, seleccione Instrumentos y luego Volver a escribir.
  2. En el Reescribir la configuración diálogo, seleccione Habilitar reescritura.
  3. Agregar un nuevo conjunto llamado Encabezados de Akamai.
  4. Agregue una nueva regla.

  5. En el Regla de reescritura cuadro de diálogo, seleccione un tipo de Agregar encabezado.

  6. En el Reemplazar sección, agregue lo siguiente:

    Nombre: Pragma

    Valor:

          akamai-x-get-request-id, akamai-x-cache-on, akamai-x-cache-remote-on,
      akamai-x-check-cacheable, akamai-x-get-cache-key, akamai-x-get-extracted-values,
      akamai-x-get-nonces, akamai-x-get-ssl-client-session-id, akamai-x-get-true-cache-key,
      akamai-x-serial-no
  7. Seleccione OK , y entonces OK.
