Ver la configuración

Siga estos pasos para ver la configuración de un reproductor:

  1. En Studio, vaya al JUGADORES módulo.
  2. Haga clic en el video cuya configuración desea ver.
  3. Haga clic en el Publicar e insertar botón.
  4. Asegúrese de que se publiquen todos los cambios.
  5. Haga clic en el URL del jugador (como se muestra a continuación) Verá que el reproductor se abre en un navegador.
  6. Verifique la URL en el navegador, aparecerá algo como esto: 
    https://players.brightcove.net/1752604059001/SAJo2wjVo_default/index.html
  7. Cambiar el index.html en la URL a config.json.
  8. La URL alterada mostrará la configuración del reproductor:
    Nota: Es posible que el JSON resultante no se muestre tan bien como se muestra en la captura de pantalla. Puede encontrar una extensión / complemento del formateador JSON para su navegador para ayudar a formatear el JSON.

Cambiar la configuración

Aunque cambiar la configuración está más allá del alcance de este documento, un par de comentarios sobre ese tema son apropiados.

  • Studio proporciona una interfaz de usuario para modificar muchos aspectos de la configuración del reproductor en el JUGADORES módulo.
  • Para un control completo de la configuración del reproductor, API de gestión de jugadores puede ser usado. Tenga en cuenta que esto es para desarrolladores y utiliza rizo declaraciones extensivamente. La Paso a paso: Gestión de jugadores El documento proporciona una introducción al proceso.
