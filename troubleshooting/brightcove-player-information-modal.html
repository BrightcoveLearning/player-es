Modal de información del jugador de Brightcove

{{ page.description }}

Ver información del jugador

Muestra la información del jugador haciendo clic derecho en el jugador y seleccionando Información del jugador , como se muestra en esta captura de pantalla.

Nota: En el móvil puede utilizar la pulsación larga para abrir el modal.

Una vez que seleccione Información del jugador Verá la ventana modal que contiene información sobre el jugador, como se muestra en esta captura de pantalla.

Nota: Solo los complementos suministrados por Brightcove que forman parte de la configuración aparecerán en la Complementos sección. Si agrega un complemento de Brightcove en el JavaScript de una página HTML, NO aparecerá en la lista. Los complementos desarrollados por el cliente, incluso si aparecen como parte de la configuración del reproductor, NO aparecerán.

Cerrar la ventana modal

Cierras el modal ya sea presionando el ESC o haciendo clic en el icono de cierre en la parte superior izquierda.

Modificar el menú contextual

Puede modificar el menú que aparece al hacer clic con el botón derecho, llamado menú contextual. Para hacer esto, debe modificar la configuración de su reproductor. Puede hacer esto de dos maneras:

  • Usando Studio's Editor JSON desde el JUGADORES módulo.
  • Una herramienta para realizar el cambio de configuración es rizo.

Aquí se muestran los dos cambios que puede realizar y cómo afectarlos.

Nota: Las declaraciones de curl a continuación utilizan la API de administración de jugadores para modificar la configuración del jugador. En las instrucciones curl verás valores de $EMAIL, $ACCOUNT_ID y $PLAYER_ID. Debe crear variables de entorno con esos nombres y asignar los valores correspondientes, o proporcionar los valores literales para las variables.

Evitar que aparezca el menú

Si desea evitar que aparezca el menú contextual de información del jugador, puede realizar una de las siguientes acciones:

Uso del editor JSON

Modifique la configuración del reproductor añadiendo la propiedad resaltada (asegúrese de guardar el cambio):

Usar rizo

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request PATCH \
  --data '{
      "context_menu": false
  }' \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/configuration
Nota: Cada vez que hagas un cambio en un reproductor usando curl, tienes que volver a publicar el reproductor. Puede hacerlo en Studio y verá que su reproductor está marcado como que necesita ser publicado en el Estado columna:

También puedes usar curl para publicar tu reproductor:

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request POST \
    https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/publish

Evitar la opción de menú Brightcove

Si desea evitar la Sobre Brightcove que aparezca la opción de menú, puede realizar una de las siguientes acciones:

Uso del editor JSON

Modifique la configuración del reproductor añadiendo la propiedad resaltada (asegúrese de guardar el cambio):

Usar rizo

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request PATCH \
  --data '{
    "context_menu": {
      "brightcove_branding": false
    }
  }' \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/configuration
Nota: Cada vez que hagas un cambio en un reproductor usando curl, tienes que volver a publicar el reproductor. Puede hacerlo en Studio y verá que su reproductor está marcado como que necesita ser publicado en el Estado columna:

También puedes usar curl para publicar tu reproductor:

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request POST \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/publish
