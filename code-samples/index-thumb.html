    Ejemplos de código

    Los ejemplos de código siguientes muestran personalizaciones y mejoras en Brightcove Player. Las muestras marcadas con asteriscos doble** dependen parcial o totalmente de las funciones de Video Cloud. También puede encontrar una lista alfabética de muestras aquí.
     

    Reproductor Outstream (solo anuncio)

    Este ejemplo proporciona funciones auxiliares de JavaScript que demuestran cómo ajustar dinámicamente el tamaño de un <div> elemento en el DOM de la página, inyectar el Brightcove Player allí, cargar y reproducir un anuncio y, a continuación, quitar el reproductor

    Desplazamiento del Reproductor a la vista con anuncios

    En el reproductor, observe que la barra de control de volumen horizontal reemplaza a la barra de control vertical predeterminada.

    Control de volumen horizontal (sólo reproductor 1.x)

    En el reproductor, una barra de control de volumen horizontal sustituye a la barra de control vertical predeterminada.

    Volumen vertical

    Establezca el control de volumen para que muestre un control deslizante vertical en lugar del diseño horizontal predeterminado.

    Indicadores de anuncio en la barra de reproducción

    Este ejemplo muestra un reproductor en el que los tres puntos de referencia del anuncio asociados al vídeo se muestran en la barra de reproducción como marcadores azules.

    Publicidad con el plugin SSAI

    Eneste ejemplo se utilizan anuncios IMA definidos en un archivo VMAP para entregar anuncios del lado del servidor en la transmisión de vídeo. Hay un anuncio de pre-roll, mid-roll y post-roll. El archivo VMAP se define en la configuración del anuncio.

    Temporizador de cuenta atrás de anuncios

    Cree un temporizador de cuenta atrás para un anuncio utilizando el plugin Overlay.

     

    Puerta de edad con ModalDialog

    Este ejemplo utiliza el ModalDialog para cubrir inicialmente el reproductor. Cuando el usuario confirma su edad haciendo clic en la ventana modal, se muestran los controles y se reproduce el vídeo.

    Visualización de puntos de referencia CTA**

    El jugador muestra dos CTA diferentes que se muestran en función de los puntos de referencia. El primer CTA se muestra a los 3 segundos, durante una duración de 5 segundos. El segundo CTA se muestra a los 13 segundos durante una duración de 3 segundos.

    Llamada dinámica**

    Podrás ver tres diferentes técnicas de visualización de llamadas a la acción con este reproductor.

    Reproducir/Pausar vídeo desde iframe Parent

    Haga clic en el botón de reproducción o pausa, que forma parte de la página principal del iframe, para verlo iniciar la reproducción/pausa del vídeo en el reproductor iframe.

    Puerta de edad

    En este ejemplo se muestra una superposición que oculta los controles del reproductor y el botón de reproducción. Cuando el usuario confirma su edad haciendo clic en la superposición, se muestran los controles y se reproduce el vídeo.

    Reproducción automática con botón Dessilenciar

    El vídeo se comportará de manera diferente dependiendo del navegador/plataforma en la que ejecute el video. Si el vídeo se reproducirá automáticamente con audio, lo hace. De lo contrario, el video comenzará a reproducirse, pero aparecerá un botón de anular el silencio.

    Lista de reproducción aleatoria de vídeo de fondo**

    En el reproductor verás vídeos seleccionados aleatoriamente de una lista de reproducción que se reproducen continuamente como vídeos de fondo. Al comienzo de cada vídeo se muestra el título y la descripción de cada vídeo.

    Lista de reproducción de vídeo de fondo**

    En el reproductor verá que hay cuatro videos que se reproducen continuamente como videos de fondo. Además, al comienzo de cada vídeo se muestran el título y la descripción de cada vídeo superponiendo el vídeo de fondo.

    Vídeo de fondo de ventana completa**

    En este tema, aprenderá a usar una lista de reproducción para proporcionar vídeos en segundo plano y cambiar automáticamente el tamaño del reproductor cuando se cambia el tamaño de la ventana del navegador para que el vídeo llene completamente la ventana del navegador.

    Creación de un bucle de vídeo

    Hay algunas situaciones en las que es posible que desee usar un video de bucle. Este ejemplo bucle el vídeo 3 veces y, a continuación, detiene la reproducción de vídeo.

    Deshabilitar el depurador de progreso

    Reproduce el video. Observe que no puede utilizar el depurador de progreso para avanzar en el vídeo.

    Detener a otros reproductores en la página cuando se inicia un vídeo

    Empieza a reproducir uno de los vídeos. A continuación, reproduzca el vídeo en otro reproductor y vea que la reproducción de vídeo se detiene en el otro reproductor que se estaba reproduciendo.

    Reproducción de velocidad variable

    Inicie la reproducción de vídeo y seleccione el botón de velocidad de reproducción en la barra de controles (el valor predeterminado es 1x). Elige una velocidad diferente para ver cómo cambia la velocidad de reproducción de vídeo.

    Cargar el reproductor dinámicamente

    En este ejemplo, haga clic en el botón Agregar reproductor para crear y cargar dinámicamente el reproductor y, a continuación, reproducir el vídeo.

    Visualización de vistas en la barra de controles

    Cuando se reproduzca el vídeo, verá las vistas de vídeo en la barra de controles. Puede haber un ligero retraso antes de que se muestre la información ya que Analytics API se consulta asincrónicamente los datos.
