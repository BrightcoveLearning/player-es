Muestras de código

En los siguientes ejemplos de código se muestran personalizaciones y mejoras de Brightcove Player. Las muestras marcadas con dos asteriscos ** dependen total o parcialmente de las funciones de Video Cloud. También puede encontrar una lista alfabética de muestras. aquí.
