Documentación de Brightcove Player

El desarrollo de Brightcove Player te permite personalizar, integrar o añadir funcionalidad a tus jugadores. Las herramientas de desarrollo son la Player API y las tecnologías web estándar de HTML5, CSS y JavaScript.

Echa un vistazo a la extensa lista de ejemplos de código para conocer las posibilidades de personalización del reproductor.
